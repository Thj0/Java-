
# Objetivo do curso

Ao final do curso, o Dev será capaz de entender o que é o Collection Framework e a aplicabilidade de uas principais interfaces: List, Set e Map.
Também aprender um pouco de Strea API e como ela pode nos ajudar na manipulação das collections.
