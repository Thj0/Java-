# Configurando o ambiente 

Java é composta por duas ferramentas, uma é a JDK e a outra JRE.

No aspecto de desenvolvedor, a gente sempre tem que considerar a instalação do JDK através do site oficial da Oracle.



Para começar a desenvolver em Java é necessário que tenhamos o JDK instalado.

**É extremamente relevante determinar a versão do Java que precisa utilizar diante de alguns requisitos do projeto.**



## Instalando o JDK no Windows 

Algumas IDEs como o [VSCode](https://code.visualstudio.com/download) já consegue baixar e instalar o JDK e realizar a configuração de forma fácil.

- Buscando no Google por Java [JDK](https://download.oracle.com/java/25/latest/jdk-25_windows-x64_bin.exe ( [sha256](https://download.oracle.com/java/25/latest/jdk-25_windows-x64_bin.exe.sha256) ) (Versão desejada)
- E por fim selecione a opção **.exe** de acordo com o seu sistema operacional. 
- Ex: x32.exe ou x64.exe

Feito isso.

- Após realizar o download, executar o instalador para instalar o Java no Windows. Esse processo instalará tanto o **JDK** quando o **JRE**. 

***Evite mudar o diretório de instalação!***

Após a instalação.

- Precisamos agora validar se a instalação também já configurou nossa variável de ambiente para poder executar o Java pelo Prompt de comando ou PowerShell do Windows.

- Abre o Prompt de comando e execute o comando java -version.
