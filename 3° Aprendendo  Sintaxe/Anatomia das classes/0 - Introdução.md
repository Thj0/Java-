# Objetivo do curso 

Apresentar as regras essenciais para a construção de códigos com base na linguagem Java.



## Pré-requisitos 

- JDK instalado.
- IDE escolhida. 
- Diretório do projeto definido.

--

***SUGESTÃO***  

Podemos separa em pacotes, diretórios ou em projetos.



### Anatomia da Classe 

**O que é?**

É basicamente consolidar todo aspecto estrutural, onde a gente aplica os códigos. 

- Porque esse código inicializou aqui e ele precisou ser encerrado em uma outra instrução dentro do mesmo arquivo?
- Porque existe um encadeamento de caracteres, de chaves, de colchetes, e assim sucessivamente?
- Porque algumas palavras são escritas de forma espaçadas?

***Que é chamado de Indentado dento do código.***



Compreendendo a estrutura inicial, a base sobre os conteúdos e algoritmos que estão sendo apresentados.



## Assuntos 

-  Estrutura inicial.

Conhecer o que é uma classe, e etc.



- Padrões de nomenclatura.

Saber que tem uma necessidade de aplicar alguns padrões, convenções e automaticamente o código vai se tornar mais legível.

E com isso entender porque desenvolvedores escreveram determinados algoritmos usando essa estrutura através de padrões de nomenclatura.



- Declarando variáveis e métodos.

O que são variáveis?

O que são métodos?

Quais são os aspectos que podemos aplicar, em todas as expressões que se referem a variáveis e métodos? 

Como é de percepção comum, existe uma estrutura universal, então todas as variáveis seguirão um padrão. Todos os métodos, também seguirão esse padrão. 

E a partir do momento que compreendemos isso, fica mais claro a nossa interação com os projetos. 



- Indentação.

O que é e o que vai ser, dentro do código.



- Organizando arquivos.

Criar uma organização através de pacotes, que nada mais é do que subdiretórios bem distribuídos. 

Sem isso, tal projeto fica ingerenciável. 



- Java Beans.

O que é esse conceito a  nível de estruturação do meu código?

O que é essa convenção, o que ela é capaz de fazer para trazer mais clareza e objetividade nas minhas implementações seguindo as particularidades e conceitos da linguagem?
