# Java Sintaxe



### Objetivo do curso 

Explorar os tipos de dados numéricos inteiros, numéricos decimais, lógicos, caracteres e suas aplicabilidades. Explicar a diferença entre definição de variáveis e constantes.

_____

## Pré-requisitos 

- JDK instalado.
- IDE escolhida.
- Diretório do projeto definido.
- Interação com o Github.

_____

# Tipos e Variáveis.



### Assuntos 

- Numéricos inteiros e decimais.
- Tipos lógicos.
- Caracteres.
- Objetos.
- Diferença entre variável e constante.
