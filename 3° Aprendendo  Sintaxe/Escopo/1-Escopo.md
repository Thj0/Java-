# Escopo 



### Objetivo do curso

Identificar a localização mais conveniente para a escrita de algoritmo necessária para o nosso programa.



E é através desse conceito de escopo, que a gente consegue ter uma melhor e/ou mais adequada estrutura das nossas implementações.

____



## Pré-requisitos

* Diretório do projeto definido.
* Interação com Github.
* IDE configurada.

____



## Assuntos 

Entender quais são os três tipos de escopos iniciais.

* Escopo de classe.

* Escopo de método.
* Escopo de fluxo.
