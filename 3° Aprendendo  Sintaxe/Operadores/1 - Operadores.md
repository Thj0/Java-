# Operadores 



No contexto de Java Sintaxe, os operadores são símbolos especiais que utilizamos para operações e expressões matemáticas.

____

## Objetivo do curso 

Apresentar o significado dos símbolos especiais de acordo com a linguagem de programação Java.

____

### Pré-requisitos 

- Diretório do projeto definido.
- Integração com Github.
- Noções em declaração de variáveis.

____

### Assuntos 

- Operadores aritméticos.
- Operadores unários.
- Operadores relacionais.
- Operadores lógicos.
- Operador ternário.
