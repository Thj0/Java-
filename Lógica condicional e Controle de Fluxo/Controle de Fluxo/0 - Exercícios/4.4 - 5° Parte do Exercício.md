

Esse switch representa o dia da semana e forma de número.

switchSemana();

```Java 
public class ExTres{

	public static void main(String[] args){ 
	
	
	}
	
	 private static void switchSemana() {

  

            String dia = "Sexta";

  

                switch (dia) {

                    case "Segunda":

                        System.out.println(2);

                        break;

                    case "Terça":

                        System.out.println(3);

                        break;

                    case "Quarta":

                        System.out.println(4);

                        break;

                    case "Quinta":

                        System.out.println(5);

                        break;

                    case "Sexta":

                        System.out.println(6);

                        break;

                    case "Sábado":

                        System.out.println(7);

                        break;

                    case "Domingo":

                        System.out.println(1);

                        break;

                    default:

                        break;

           }

     }

}
```

Dado o nome do dia da semana, informe o número que o representa.

Usar o switch nesse caso é muito melhor.

Porque aqui, temos somente uma variável e queremos saber quais os valores que ela pode assumir.
