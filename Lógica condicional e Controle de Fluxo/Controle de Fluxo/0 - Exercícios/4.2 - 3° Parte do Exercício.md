Exemplo de como não se fazer um if 

Como temos somente uma variável sobre o if, nesse caso deveríamos usar o switch. 

Porque temos um valor exato, e como dito, tem apenas uma variável sendo analisada.

É a forma de decisão que se adequa ao que o método está propondo.
___

Com o if não seria a melhor forma de resolver porque queremos saber quais são os valores possíveis que aquela variável consegue ter.

E podem ser muitos. Então ao invés de fazer um if gigante, é melhor utilizar um switch.

___

**Resumo**

Se o meu teste envolve apenas uma variável, ou seja, é um valor exato que quero descobrir; Então é melhor eitar o uso do IF e utilizar o switch.
