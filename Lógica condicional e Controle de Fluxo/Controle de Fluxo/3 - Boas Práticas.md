### Exercitando

Criar um simples projeto com variáveis e expressões para usar nas estruturar **if** e **switch**.

Com if, exiba o nome do mês do ano de acordo com seu número.
Evite o efeito "flecha". 

Faça também outro if que verifique se o mês é julho, dezembro ou janeiro, para assim exibir o texto "Férias".


Com **Switch**, use String para a partir do dia da semana, exibir seu número. 

Ainda no switch, faça outro exemplo onde, se uma variável inteira for entre 1 e 3 exibir o texto "Certo". Se for 4 exibir "Errado", se for 5 "Talvez". 

Para demais valores exibir "Valor indefinido".
