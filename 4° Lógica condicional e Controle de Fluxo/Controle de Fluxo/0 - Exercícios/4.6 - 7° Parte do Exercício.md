```Java
public class ExTres { 


	public static void main(String[] args){ 
	
	}
	

    private static void switchFerias() {

        String mes = "Dezembro";

  

        switch (mes) {

            case "Dezembro":

            case "Julho":

            case "Janeiro":

                System.out.println("Férias");

                break;

  

            default:

                System.out.println("Mês indefinido");

                break;

        }

    }
    
}
```

Esse exercíciosegue a mesma ideia do exemplo passado, mas ao invés de um valor inteiro(numérico), se trata de um texto(String).
