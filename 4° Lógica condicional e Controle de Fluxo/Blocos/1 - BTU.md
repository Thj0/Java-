Blocos, Tipos e Uso

___

### Objetivos 

1. Entender o que são blocos 
2. Tipos de blocos
3. Saber com usar


___

### Conceituação 

"É um grupo de 0 ou mais códigos quais trabalham em conjunto para executar uma operação".


**Tipos: 
 - Locais: dentro de métodos
 - Estáticos: dentro de classes 
 - Instância: dentro de classes

___

### Criação

Tipos
 - Locais 
{
...
}

Tudo que estiver dentro dessas chaves, faz parte do bloco de execução.

___
### Exemplos 


if (autoeizado) { 
	CarregarPerfil
	DirecionarPáginaPrincipal
}

if (menorIdade)
	DirecionarPáginaProibido


if (autorizado) { 
...
} else { 
...
}
